# Rahul Shewatkar — Portfolio Website

Simple one‑page dark portfolio (HTML/CSS/JS). Open locally, or deploy to GitHub Pages / Netlify / Vercel.

## Quick start (local)
- Open `portfolio-website/index.html` in your browser.

## Add your photo + resume
Place files into `portfolio-website/assets/`:
- `profile.jpg`
- `Rahul_Shewatkar_Resume.pdf`

The site already links to `./assets/Rahul_Shewatkar_Resume.pdf`.

## Customize content
Edit `index.html`:
- Headline / summary in the Hero section
- Experience bullets
- Projects (add new cards + links)

## Deploy
### GitHub Pages
1. Create a GitHub repo (e.g. `portfolio-website`)
2. Upload all files in this folder
3. In GitHub repo settings → Pages → Deploy from branch → select branch and root folder

### Netlify / Vercel
- Drag-and-drop this folder into the dashboard, or connect the repo.
